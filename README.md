# 猫咪小窝

猫咪小窝是一个以猫咪为主题的温馨可爱网页项目，旨在为用户提供一个直观且具有吸引力的界面来探索计算机专业的不同方向。该项目包含首页、方向列表以及工程路线图等多个页面。

## 功能特点

- **响应式设计**：适配不同设备屏幕大小，提供良好的浏览体验。
- **清晰导航**：通过首页可以轻松进入各个计算机专业方向。
- **路线图展示**：提供前端与后端工程学习路线的树状图展示。
- **精美动画**：多层次的入场动画和交互动画，提升用户体验。
- **固定山水背景**：使用精美的山水背景图，固定在视口，提供优雅的视觉体验。

## 🎬 动画效果说明

### Tea Roadmap 页面动画

本项目的 `tea-roadmap.html` 页面实现了多层次的入场动画效果，创造出流畅、优雅的视觉体验。

#### 动画时间轴

**页面加载时（0-2.5秒）**

1. **顶部装饰条** (0s - 1.2s)
   - 从顶部滑入，持续1.2秒
   - 效果：从页面顶部-100%位置滑入到正常位置

2. **导航栏** (0s - 1.2s)
   - 从顶部滑入，持续1.2秒
   - 与装饰条同步，从顶部滑入并淡入

3. **背景图片缩放** (0s - 1.6s)
   - 持续1.6秒的缩放动画
   - 背景从110%缩放到100%，同时位置从顶部移动到20%位置

4. **云雾散开效果** (0s - 2.5s)
   - 第一层云雾：2秒内散开（延迟0s）
   - 第二层云雾：2.5秒内散开（延迟0.3s）
   - 云雾向外扩散并上移，同时透明度降至0

5. **标题文字** (0.3s - 1.5s)
   - 延迟0.3秒，持续1.2秒
   - 从底部100px位置向上滑入并淡入

6. **描述文字** (0.5s - 1.7s)
   - 延迟0.5秒，持续1.2秒
   - 从底部100px位置向上滑入并淡入

7. **茶叶装饰** (0.8s - 2.5s)
   - 左侧茶叶：延迟0.8秒
   - 右侧茶叶：延迟1秒
   - 从0.5倍缩放到正常大小，淡入至10%透明度，然后开始漂浮动画

#### 滚动触发动画

- **章节标题**
  - 触发时机：元素进入视口15%时
  - 持续时间：0.8秒
  - 效果：从底部100px位置滑入并淡入

- **卡片动画**
  - 触发时机：元素进入视口15%时
  - 持续时间：0.6秒
  - 延迟：每张卡片间隔0.05秒（形成波浪效果）
  - 效果：从底部50px位置滑入并淡入

#### 背景效果

- **固定山水背景**
  - 使用 `mountain-bg.jpg` 作为固定背景
  - 位置：固定在视口，z-index: 1
  - 效果：页面加载时立即显示
  - 全屏显示，覆盖整个视口
  - 背景缩放动画（1.6秒）
  - 性能优异，GPU加速

#### 交互动画

- **语言切换动画** (0.7秒)
  - 内容先缩小淡出（0.3秒）
  - 切换语言文本
  - 内容放大淡入（0.4秒）
  - 按钮有圆形扩散效果
  - 支持中英文无缝切换

- **返回按钮动画** (0.8秒)
  - 所有入场动画逆向播放
  - 顶部导航向上滑出
  - Hero区域背景缩小
  - 内容从底部退出
  - 装饰元素缩小消失
  - 完成后跳转到目标页面

- **响应式适配**
  - 桌面端：返回按钮在导航栏左侧
  - 移动端：返回按钮在菜单内，居中显示
  - 语言切换按钮在移动端集成到菜单中

#### 动画关键帧

| 动画名称 | 用途 | 效果描述 |
|---------|------|---------|
| `topIn` | 顶部导航入场/退出 | 从-100%位置滑入，透明度0→1（支持逆向） |
| `backgroundZoom` | 背景图片入场/退出 | 从110%缩放到100%，位置调整（支持逆向） |
| `bottomInText` | 文字内容入场/退出 | 从底部100px滑入，透明度0→1（支持逆向） |
| `fadeInUp` | 卡片元素 | 从底部50px滑入，透明度0→1 |
| `fadeOutScale` | 内容切换退出 | 缩小至95%，透明度降至0 |
| `fadeInScale` | 内容切换进入 | 从95%放大至100%，透明度0→1 |
| `pageExit` | 页面退出 | 向下移动50px，透明度降至0 |
| `zoomInDecoration` | 装饰元素 | 从0.5倍缩放到1倍，透明度至0.1（支持逆向） |
| `disperseMist1/2` | 云雾散开 | 放大1.5-1.8倍并上移，透明度降至0 |
| `float` | 漂浮效果 | 上下移动20px并旋转10度 |

#### 技术实现

- **IntersectionObserver API**：用于滚动触发动画
  - 阈值：15%（元素进入视口15%时触发）
  - 根边距：底部-100px（提前触发动画）
  - 单次触发：避免重复播放

- **性能优化**
  - 使用 CSS 动画（GPU 加速）
  - 使用 `transform` 和 `opacity` 属性
  - 动画只触发一次，避免重复计算

#### 动画层次结构

```
页面层次（从后到前）：
1. 背景渐变 + 山景图片（带缩放动画）
2. 云雾层（散开动画）
3. Hero 内容（文字入场动画）
4. 导航栏（顶部滑入）
5. 茶叶装饰（缩放+漂浮）
6. 卡片内容（滚动触发）
```

#### 自定义调整

**修改动画速度**：
```css
animation: bottomInText 1.2s ease-out 0.3s forwards;
/*                      ^持续   ^延迟        */
```

**修改动画延迟**：
```css
.hero-content h1 {
    animation: bottomInText 1.2s ease-out 0.3s forwards;
    /*                                     ^调整延迟   */
}
```

## 安装指南

此项目是纯静态网页项目，无需复杂的安装步骤。只需将项目文件放置于Web服务器上即可访问。

## 使用说明

1. **首页浏览**
   - 打开浏览器访问首页，您将看到一个温馨的猫咪主题界面
   - 点击相应的链接或按钮，可以进入不同的计算机专业方向页面

2. **路线图页面**
   - 在 `tea-roadmap.html` 页面，您可以查看完整的开发者学习路线
   - 页面加载时会自动播放多层入场动画（约2.5秒）
   - **固定山水背景**：
     - 使用精美的山水背景图（mountain-bg.jpg）
     - 背景固定在视口，不随滚动移动
     - 页面加载时有缩放动画效果

3. **语言切换**
   - 点击右上角的 `EN` 或 `中文` 按钮切换语言
   - 切换时会有内容缩放淡入淡出的过渡动画（0.7秒）
   - 所有文本内容会平滑切换到对应语言

4. **返回首页**
   - 桌面端：点击导航栏左侧的"返回首页"或"Back"按钮
   - 移动端：打开菜单，点击菜单内的返回按钮
   - 返回时会播放逆向退出动画（0.8秒），然后跳转

5. **响应式体验**
   - 在移动设备上，点击右上角的 `☰` 图标打开菜单
   - 菜单内包含所有导航链接、返回按钮和语言切换选项
   - 自动适配不同屏幕尺寸
